## 📝 Requirements

**Nodejs v16.8.0**

Set node version with _nvm_ previously installed, by running:

```bash
nvm use
```

This will get the version specified on the .nvmrc file.

## 🛠 Installation

Use npm to install dependencies

```bash
npm install
```

## 🕷 How to run the scraper 🕷

Just execute

```bash
npm start
```
